# PingOps Node.js SDK

OpenTelemetry-compatible Node.js SDK for monitoring outgoing API calls.

## Packages

- `@pingops/core` - Internal shared utilities
- `@pingops/otel` - OpenTelemetry `SpanProcessor` implementation
- `@pingops/sdk` - User-facing API for bootstrapping and manual instrumentation

## Installation

```bash
pnpm install
pnpm build
```

## Development

```bash
# Build all packages
pnpm build

# Clean all packages
pnpm clean
```
